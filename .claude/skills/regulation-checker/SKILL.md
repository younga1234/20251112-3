# Regulation Checker - 규정 준수 체크리스트

## 목적
국가유산청(구 문화재청)의 「발굴조사의 방법 및 절차 등에 관한 규정」과 「매장문화재 발굴조사업무 처리지침」에 근거하여 발굴조사 보고서 및 고찰 섹션의 필수 구성 요소 체크리스트를 생성하고, 현재 보유 자료 대비 결손 항목을 식별합니다.

## 입력
- `output/normalized/metadata.csv`: 수집된 자료 목록
- `output/compare/nearby_ranked.csv`: 주변 유사 유적 목록
- **규정 문서 (내장)**:
  - 발굴조사의 방법 및 절차 등에 관한 규정
  - 매장문화재 발굴조사업무 처리지침
  - 국가유산 협업포털 보고서 작성 지침

## 처리 프로세스

### 1단계: 보고서 전체 구성 체크리스트
「발굴조사의 방법 및 절차 등에 관한 규정」 제10조(보고서의 작성) 기준:

```markdown
## 발굴조사 보고서 필수 구성 항목

### A. 행정사항
- [ ] A1. 발굴조사 허가번호 및 허가일
- [ ] A2. 조사기간 (시작일~종료일)
- [ ] A3. 조사기관 및 조사단 구성
- [ ] A4. 조사원 명단 (직책, 성명, 담당 분야)
- [ ] A5. 발굴면적 및 조사구역 범위
- [ ] A6. 사업자 및 사업명

### B. 조사 배경 및 경과
- [ ] B1. 조사에 이르게 된 경위
- [ ] B2. 조사 목적 및 필요성
- [ ] B3. 조사 방법 및 절차
- [ ] B4. 조사 진행 경과 (일지 요약)
- [ ] B5. 관련 회의 및 자문 내용

### C. 유적의 입지 및 환경
- [ ] C1. 지리적 위치 (주소, 좌표)
- [ ] C2. 지형 및 지질
- [ ] C3. 주변 환경 (하천, 산, 평야 등)
- [ ] C4. 토지이용 현황
- [ ] C5. 기후 및 식생

### D. 역사적 배경
- [ ] D1. 문헌 기록 검토
- [ ] D2. 지명 유래 및 변천
- [ ] D3. 주변 유적 분포 현황
- [ ] D4. 이전 조사 사례 검토

### E. 조사 내용
- [ ] E1. 층위 및 토층 설명
- [ ] E2. 유구 설명 (개별 유구별 상세 기술)
  - [ ] E2-a. 유구 번호, 위치, 규모
  - [ ] E2-b. 평면 형태 및 단면 형태
  - [ ] E2-c. 출토 유물 목록
  - [ ] E2-d. 층위 관계 (중복, 선후 관계)
- [ ] E3. 유물 설명 (종류별 분류 및 특징 기술)
  - [ ] E3-a. 토기 (형식, 문양, 제작 기법)
  - [ ] E3-b. 석기 (석재, 형태, 용도)
  - [ ] E3-c. 금속유물 (재질, 형태, 기능)
  - [ ] E3-d. 기타 유물
- [ ] E4. 특수 유물 또는 유구에 대한 상세 분석

### F. 과학적 분석 (해당 시)
- [ ] F1. 방사성탄소연대측정 (C14)
- [ ] F2. 토양 분석
- [ ] F3. 식물유체 분석 (종자, 화분 등)
- [ ] F4. 동물유체 분석 (뼈, 패각 등)
- [ ] F5. 금속 성분 분석
- [ ] F6. 석재 산지 분석

### G. 도면 및 사진
- [ ] G1. 유적 위치도 (1/5,000 ~ 1/25,000)
- [ ] G2. 유구 배치도 (전체 평면도)
- [ ] G3. 유구 실측도 (개별 평·단면도)
- [ ] G4. 유물 실측도
- [ ] G5. 조사 전·중·후 전경 사진
- [ ] G6. 유구 사진 (조사 중, 완료 후)
- [ ] G7. 유물 사진 (개별, 집합)
- [ ] G8. 층위 사진

### H. 고찰 (본 스킬의 주요 대상)
- [ ] H1. 조사 성과 종합
- [ ] H2. 유구의 성격 및 기능 해석
- [ ] H3. 유물의 편년 및 계통
- [ ] H4. 주변 유적과의 비교 분석
- [ ] H5. 공간 구조 및 취락(유적) 구조 해석
- [ ] H6. 역사·고고학적 의의
- [ ] H7. 보존 및 활용 방안
- [ ] H8. 향후 과제 및 제언

### I. 결론
- [ ] I1. 조사 결과 요약
- [ ] I2. 학술적·사회적 가치 평가
- [ ] I3. 마무리 및 감사의 글

### J. 부록
- [ ] J1. 유구 일람표
- [ ] J2. 유물 관찰표
- [ ] J3. 참고문헌 목록
- [ ] J4. 영문 초록 (Abstract)
- [ ] J5. 조사일지 (선택)
- [ ] J6. 회의록 및 자문 의견서 (선택)
```

### 2단계: 고찰 섹션 상세 체크리스트
고찰(考察)은 보고서의 핵심으로, 다음 8개 구성 요소를 포함해야 합니다:

```markdown
## 고찰(考察) 섹션 필수 구성 요소

### H1. 조사 성과 종합
- [ ] H1-1. 조사된 유구의 종류와 수량 요약
- [ ] H1-2. 출토 유물의 종류와 수량 요약
- [ ] H1-3. 주요 발견사항 강조
- [ ] H1-4. 조사의 한계 및 미진한 부분 언급

**근거 자료 확인**:
- [ ] 유구 일람표 존재
- [ ] 유물 관찰표 존재
- [ ] 수치 데이터 정확성

### H2. 유구의 성격 및 기능 해석
- [ ] H2-1. 유구 유형별 성격 해석 (주거, 저장, 제사, 생산 등)
- [ ] H2-2. 유구 간 기능적 관계 분석
- [ ] H2-3. 유구 배치 패턴 및 공간 구조 해석
- [ ] H2-4. 유구 사용 시기 및 폐기 과정 추정

**근거 자료 확인**:
- [ ] 유구 평·단면도
- [ ] 출토 유물과 유구의 연관성 분석
- [ ] 층위 관계 명확성

### H3. 유물의 편년 및 계통
- [ ] H3-1. 유물의 형식 분류 및 편년
- [ ] H3-2. 절대연대 자료 제시 (C14 등)
- [ ] H3-3. 형태적·기술적 계통 분석
- [ ] H3-4. 제작 기법 및 원료 산지 추정

**근거 자료 확인**:
- [ ] 유물 실측도 및 사진
- [ ] 형식학적 비교 근거
- [ ] 과학적 분석 결과 (해당 시)

### H4. 주변 유적과의 비교 분석 ⭐ (similarity-matcher 연계)
- [ ] H4-1. 비교 대상 유적 선정 기준 명시
- [ ] H4-2. 공간적 분포 비교 (거리, 입지)
- [ ] H4-3. 시대적 동시성 또는 선후 관계 검토
- [ ] H4-4. 유구·유물 형태 비교
- [ ] H4-5. 공통점과 차이점 도출
- [ ] H4-6. 지역 간 교류 및 문화 전파 양상

**근거 자료 확인**:
- [ ] 주변 유적 목록 (output/compare/nearby_ranked.csv)
- [ ] 비교표 또는 대조 도면
- [ ] 인용 문헌 명확성

### H5. 공간 구조 및 취락(유적) 구조 해석
- [ ] H5-1. 전체 유적의 공간 구성 원리
- [ ] H5-2. 중심-주변 구조 또는 군집 분석
- [ ] H5-3. 생활 공간, 의례 공간, 생산 공간 구분
- [ ] H5-4. 취락 규모 및 인구 추정 (가능 시)
- [ ] H5-5. 방어 시설 또는 경계 요소 분석

### H6. 역사·고고학적 의의
- [ ] H6-1. 해당 지역 고고학 연구사에서의 위치
- [ ] H6-2. 새로운 발견 또는 기존 학설 보강/반박
- [ ] H6-3. 문헌 기록과의 대조 (해당 시)
- [ ] H6-4. 한국 고고학 전반에 대한 기여
- [ ] H6-5. 국제적 비교 연구 가능성 (해당 시)

### H7. 보존 및 활용 방안
- [ ] H7-1. 유적의 보존 상태 평가
- [ ] H7-2. 보존 처리 방안 제안
- [ ] H7-3. 현지 보존 vs 이전 보존 검토
- [ ] H7-4. 교육·관광 자원으로의 활용 방안
- [ ] H7-5. 지역 사회와의 연계 방안

### H8. 향후 과제 및 제언
- [ ] H8-1. 미해결 문제점 제시
- [ ] H8-2. 추가 조사 필요 영역
- [ ] H8-3. 후속 연구 방향 제안
- [ ] H8-4. 학계에 대한 제언
```

### 3단계: 현재 보유 자료 대비 결손 분석
수집된 자료(metadata.csv, nearby_ranked.csv)를 바탕으로 체크리스트 각 항목의 충족 여부를 평가:

```python
def assess_completeness(metadata_csv, nearby_csv):
    """자료 완전성 평가"""
    gaps = []

    # 주변 유적 비교 자료 확인
    nearby_count = len(load_csv(nearby_csv))
    if nearby_count < 5:
        gaps.append({
            'item': 'H4. 주변 유적과의 비교 분석',
            'status': '불충분',
            'current': f'{nearby_count}개 유적',
            'required': '최소 5~10개 유적',
            'action': '추가 유적 자료 수집 또는 비교 반경 확대'
        })

    # 과학적 분석 자료 확인
    has_c14 = check_keyword_in_docs('방사성탄소', 'C14', 'radiocarbon')
    if not has_c14:
        gaps.append({
            'item': 'F1. 방사성탄소연대측정',
            'status': '없음',
            'current': '자료 없음',
            'required': '최소 1개 시료 분석 결과',
            'action': '연대 측정 자료 보강 또는 형식학적 편년으로 대체'
        })

    # 도면 자료 확인
    has_drawings = check_files('*.dwg', '*.pdf', '도면')
    if not has_drawings:
        gaps.append({
            'item': 'G. 도면 및 사진',
            'status': '확인 필요',
            'current': '도면 파일 미확인',
            'required': '유구 배치도, 실측도 필수',
            'action': '도면 파일 추가 또는 조사단에 요청'
        })

    return gaps
```
