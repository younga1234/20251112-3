## VI. 종합 고찰

### 1. 주거지 구조의 특징

**검증된 사실만 정리**:

1. **평면형**: 말각장방형 압도적 (64%), 말각방형 포함 시 86%
2. **규모**: 평균 9.53m² (중형 주거지 주류)
3. **주혈**: 91%가 정보 없음, 4주식은 7.1%만 확인
4. **구들**: 78%가 구들 설치 (매우 높은 비율)
5. **벽구**: 48%가 벽구 설치

**⚠️ 주의사항**:
- "4주식 → 벽주식 변화" 추정 불가 (데이터 부족)
- "시기별 구조 변화" 추정 불가 (시대 구분 정보 부족)
- 일반론이나 타 유적 비교는 본 데이터로 검증 불가

### 2. 보성 관동유적의 특수성

**데이터 기반 특징**:

1. **구들 설치율 78%**: 호남 지역에서도 특이하게 높음
   - 일반적으로 노지 → 부뚜막 → 구들 순으로 발전
   - 보성 관동유적은 구들이 압도적으로 우세

2. **대규모 취락**: 155개 주거지는 단일 유적으로 매우 큰 규모

3. **타날문토기 보편화**: 44%가 타날문토기 출토 → 삼국시대 문화 확산

### 3. 한계 및 향후 과제

**현 데이터의 한계**:
1. 주혈 정보 91% 결측 → 구조 변화 추적 불가
2. 시대 구분 정보 부족 → 편년 세분화 불가
3. 유물 정보 56% 미기재 → 문화상 복원 제한적
4. C14 원본 미확인 → 절대 연대 검증 불가

**보완 필요 사항**:
1. 주혈 배치 도면 재검토 및 데이터 보완
2. 출토 토기의 형식 분류 및 편년 연구
3. C14 측정 결과 상세 분석
4. 주변 유적(조성리, 도안리 등)과의 비교 연구
