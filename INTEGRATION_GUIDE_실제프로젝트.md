## 🚀 실행 방법

### 방법 1: 기존 데이터와 스킬 통합 (추천)

#### 1단계: 저장소 클론 또는 병합
```bash
# 옵션 A: 새 폴더에서 시작
cd ~
git clone https://github.com/younga1234/up.git
cd up

# 옵션 B: 현재 위치에 데이터 가져오기
git remote add upstream https://github.com/younga1234/up.git
git fetch upstream
git merge upstream/main --allow-unrelated-histories
```

#### 2단계: 스킬 폴더 복사
```bash
# 현재 프로젝트의 스킬을 실제 데이터 저장소로 복사
cp -r /home/user/20251112-1/.claude ~/up/
```

#### 3단계: 자료 폴더 구조 확인 및 조정
```bash
cd ~/up

# 현재 구조 확인
ls -la

# 필요한 폴더 생성 (없을 경우)
mkdir -p 논문 발굴조사보고서 output

# 기존 자료 재배치
# 장흥주변유적/ 폴더가 있으면 → 주변유적/ 또는 발굴조사보고서/로 이동/복사
cp -r 장흥주변유적/* 주변유적/ 2>/dev/null || mkdir -p 주변유적 && cp -r 장흥주변유적/* 주변유적/

# data/ 폴더 내용 확인 후 적절히 분류
# 예: PDF 파일들을 논문/, 발굴조사보고서/로 분류
```

#### 4단계: 자료 분류
```bash
# data/ 폴더 내용 확인
find data/ -type f -name "*.pdf" | head -10

# 논문과 보고서 구분:
# - 논문: 학술지 논문, 학위논문
# - 발굴조사보고서: ○○유적 발굴조사 보고서
# - 주변유적: 장흥 주변 지역 자료

# 예시: 자동 분류 스크립트
for file in data/*.pdf; do
  if [[ "$file" =~ "논문|학위|연구" ]]; then
    cp "$file" 논문/
  elif [[ "$file" =~ "발굴|조사|보고서" ]]; then
    cp "$file" 발굴조사보고서/
  else
    cp "$file" 주변유적/
  fi
done
```

### 주변 유적 범위
장흥군은 전라남도 남해안에 위치하므로:
- **기본 반경**: 30km (보성, 강진, 영암, 고흥 포함)

### 시대별 특성
```yaml

  - 백제-마한 문화권
  - 고대 교통로 (영산강 수계)

  - 남해안 해상 교류
```
